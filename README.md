# SleepyPC

Нативная десктопная утилита для Windows 11, позволяющая запланировать автоматическое выключение, перезагрузку или переход ПК в спящий режим.

## Возможности

- **Выбор действия**: выключение, сон, перезагрузка
- **Два режима таймера**: обратный отсчёт (через N минут) или точное время (например, 23:45)
- **Пресеты**: быстрые кнопки на 15 мин, 30 мин, 1 час, 2 часа
- **Системный трей**: приложение сворачивается в трей при закрытии окна, показывает оставшееся время
- **Уведомление за 1 минуту**: всплывающее окно с возможностью отложить на 15 минут
- **Светлая и тёмная тема**: мягкий молочно-белый дизайн и строгая тёмная тема в стиле Windows 11 Fluent Design
- **Плавные анимации**: переходы между экранами, hover-эффекты

## Технологии

- .NET 8 / WPF
- Без сторонних NuGet-пакетов
- Кастомные стили и ControlTemplates
- System.Windows.Forms.NotifyIcon для системного трея

## Сборка и запуск

```bash
dotnet build
dotnet run
```
